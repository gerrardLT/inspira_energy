# User Stories — Inspira Energy 官网（一期）

> 产品：Inspira Energy 官网 + 业务平台 MVP
> PRD 参考：`docs/PRD-Inspira-Energy-Website.md`
> 技术栈：Next.js + Tailwind CSS + Mapbox + D3.js + Supabase + SendGrid

---

## F1 — 基金产品卡片化展示

### US-01：浏览四类基金产品概览

**描述：** 作为机构投资人，我希望在基金页面一目了然地看到四类基金（债权/股权/投建/COD）的卡片摘要，以便快速了解 Inspira 的产品体系。

**验收标准：**
1. 基金页面以网格布局展示四张卡片，每张卡片包含：基金名称、一句话描述、预期收益区间、风险等级（低/中/高）、可投地区标签、"了解更多"按钮
2. 每张卡片在桌面端为四列并排，平板两列，移动端单列堆叠
3. 预期收益区间文字下方有合规标注："*非承诺收益，仅供参考"
4. 卡片支持悬停效果（阴影/边框变化）增强交互感

---

### US-02：查看基金详情抽屉页

**描述：** 作为机构投资人，我希望点击"了解更多"后能在不离开基金页的情况下展开详情，深入了解每类基金的完整说明。

**验收标准：**
1. 点击"了解更多"后，右侧或底部滑出抽屉面板，展示该基金的详细描述、资金流向说明、适合的投资人类型、最低投资额、锁定期
2. 抽屉面板可通过关闭按钮、点击遮罩层或按 ESC 键关闭
3. 抽屉内容支持中英文切换（与当前语言一致）
4. 抽屉内包含"提交投资意向"CTA 按钮，跳转至 F2 表单

---

### US-03：按地区筛选基金产品

**描述：** 作为机构投资人，我希望能按感兴趣的市场（东南亚/中国/欧洲/澳洲）筛选出对应的基金产品，减少信息噪音。

**验收标准：**
1. 基金页面顶部提供地区筛选 Tab 或下拉选择器
2. 选择地区后，仅显示该地区的可用基金卡片，其他地区卡片隐藏
3. 默认显示"全部"，即四张卡片全部展示
4. 筛选操作无页面刷新，动画过渡平滑（fade in/out）

---

## F2 — LP 投资意向预登记表单

### US-04：提交投资意向

**描述：** 作为机构投资人，我希望在官网快速提交投资意向，让 Inspira 团队了解我的偏好并主动联系我。

**验收标准：**
1. 表单字段包含：姓名（必填）、机构名称（必填）、职位（选填）、邮箱（必填，格式校验）、电话（选填）、感兴趣基金类型（多选）、目标地区（多选）、投资规模偏好（单选下拉：$100万-$1000万 / $1000万-$5000万 / $5000万以上）
2. 必填字段未填时提交，对应字段显示红色错误提示，页面不跳转
3. 提交成功后显示感谢页面，包含"我们会在 24 小时内与您联系"文案
4. 提交后数据写入 Supabase，并触发邮件通知给 Inspira 团队（SendGrid）
5. 同一邮箱 24 小时内重复提交，显示"您已提交过，如需修改请联系我们"提示

---

### US-05：从基金详情直接跳转意向表单并预填基金类型

**描述：** 作为机构投资人，我希望从基金详情抽屉点击"提交投资意向"时，表单已自动预填对应的基金类型，减少重复操作。

**验收标准：**
1. 从抽屉页点击 CTA 按钮，跳转至投资意向表单，基金类型字段已自动选中当前浏览的基金
2. 预填字段仍可手动修改
3. URL 包含 query 参数（如 `?fund_type=debt`），支持直接访问带预填的表单页
4. 其他字段（姓名、邮箱等）保持空白，需用户填写

---

## F3 — 基金结构可视化解释器

### US-06：查看资金流向动态图解

**描述：** 作为机构投资人，我希望通过动态图解理解资金如何从 LP 流向基金再到具体项目，建立对基金结构的直观信任感。

**验收标准：**
1. 基金页面底部或独立区块展示 SVG/Canvas 动态图解，节点包含：LP → Inspira 基金 → 项目（光伏/储能/风电）
2. 资金流向用箭头动画表示，动画循环播放，视觉流畅
3. 提供"债权/股权/投建/COD"四个 Tab 切换，切换后图解内容（节点、路径、风险收益曲线）实时更新
4. 图解在桌面端宽度不小于 800px，移动端转为垂直堆叠静态版本
5. 图解加载时间 < 1 秒，动画帧率 ≥ 30fps

---

### US-07：查看风险收益对比曲线

**描述：** 作为机构投资人，我希望在同一界面看到四类基金的风险收益曲线对比，以便根据我的风险偏好快速筛选。

**验收标准：**
1. 图解右侧或下方展示折线图，X 轴为风险等级（低→高），Y 轴为预期收益区间
2. 四条曲线分别代表四类基金，颜色与卡片主题色一致，有图例说明
3. 鼠标悬停某条曲线时显示 tooltip，含该基金的预期收益范围和风险说明
4. 图表响应式：移动端图表高度自适应，可横向滚动

---

## F4 — 路条信息收集表单

### US-08：开发商分步提交路条信息

**描述：** 作为新能源开发商，我希望通过简洁的分步流程提交路条信息，避免一次性填写过多字段导致放弃。

**验收标准：**
1. 表单分 3 步，每步 ≤ 3 个字段，顶部显示进度条（Step 1/3、2/3、3/3）
   - Step 1：公司名称、联系人、邮箱
   - Step 2：地区（下拉）、项目类型（单选：光伏/储能/风电）、容量 MW（数字）
   - Step 3：当前阶段（单选：路条/可研/核准/在建）、预计投建时间（日期选择）、补充说明（文本域）
2. 每步点击"下一步"才校验当前步骤字段，错误提示显示在对应字段下方
3. 支持"上一步"回退修改，已填内容不丢失
4. 提交成功后显示确认页面，含"48 小时内专人联系"承诺

---

### US-09：上传路条支持文件

**描述：** 作为新能源开发商，我希望在提交路条时能同时上传可研报告、批文等支持文件，让 Inspira 更全面评估项目。

**验收标准：**
1. Step 3 提供文件上传区域，支持拖拽上传和点击选择
2. 支持格式：PDF、DOC、DOCX、JPG、PNG，单文件最大 10MB，最多上传 5 个文件
3. 上传进度条显示，上传失败时显示重试按钮
4. 文件存储至 Supabase Storage（私有 bucket），仅 Inspira 团队可查看
5. 文件上传为可选操作，不上传也可完成提交

---

### US-10：收到路条提交确认邮件

**描述：** 作为新能源开发商，我希望提交路条后收到一封确认邮件，确认信息已收到并了解后续流程。

**验收标准：**
1. 提交成功后 5 分钟内，开发商收到 SendGrid 发送的确认邮件
2. 邮件包含：公司名称、提交日期、路条摘要（地区/类型/容量）、Inspira 联系方式、后续流程说明
3. 邮件设计使用 Inspira 品牌模板，支持中英文（根据提交时语言自动匹配）
4. 同时触发内部通知邮件给 Inspira 项目团队，含完整提交信息和文件下载链接

---

## F5 — 交互式全球市场地图

### US-11：在首页查看 Inspira 全球业务布局

**描述：** 作为首次访问官网的用户，我希望在首页直观看到 Inspira 在哪些地区有业务活动，建立对公司规模的信任感。

**验收标准：**
1. 首页 Hero 区域下方展示交互式世界地图，覆盖东南亚、澳洲、中国、欧洲四个主要区域
2. 每个区域用光点标注，光点颜色区分状态：绿色=已布局，橙色=在谈，灰色=意向
3. 地图默认深色背景（科技感），光点有微动效（呼吸灯效果）
4. 地图组件在页面加载时 < 2 秒完成渲染

---

### US-12：点击地图区域查看地区详情

**描述：** 作为投资人，我希望点击某个地区光点后能查看该地区的项目状态和业务介绍，深入了解具体布局。

**验收标准：**
1. 点击光点后，地图旁或下方弹出地区详情卡片，包含：地区名称、状态标签、项目数量、代表项目类型（如"越南光伏 2 个在建"）
2. 详情卡片有关闭按钮，关闭后地图恢复初始状态
3. 移动端点击光点后，详情以底部抽屉形式弹出
4. 每个地区展示的项目数量和状态数据由 Supabase 读取，支持后台更新

---

## F6 — 团队背书页

### US-13：查看核心团队成员介绍

**描述：** 作为机构投资人，我希望在 About 页面了解 Inspira 核心团队的行业背景和项目经历，评估团队实力。

**验收标准：**
1. 团队页面以卡片网格展示核心成员，每张卡片包含：照片、姓名、职位、从业年限、核心能力标签（2-3 个）
2. 点击卡片展开成员详情页，含完整项目经历（时间线形式）、代表项目名称和规模
3. 照片使用真实头像（占位图开发时使用），圆形裁切，150×150px 最小尺寸
4. 卡片在桌面端三列，移动端单列

---

### US-14：查看公司里程碑时间线

**描述：** 作为访客，我希望看到 Inspira 的重要里程碑（成立、首笔投资、项目落地等），了解公司发展轨迹。

**验收标准：**
1. 团队页面下方展示垂直时间线，按时间顺序列出里程碑事件
2. 每个事件包含：时间、标题、一句话描述、可选配图
3. 时间线支持滚动触发动画（事件逐个淡入）
4. 数据从 Supabase 读取，后台可维护更新

---

## F7 — 中英双语切换

### US-15：在任意页面切换中英文

**描述：** 作为中国出海开发商或国际机构投资人，我希望随时切换官网语言，用熟悉的语言理解内容。

**验收标准：**
1. 页面右上角导航栏显示语言切换按钮（EN / 中），点击后 0.5 秒内完成切换
2. 切换后保持当前页面 URL（仅 query 参数变化，如 `?lang=en`），不跳转首页
3. 所有页面内容（导航、正文、按钮、表单选项）均有对应语言版本，无遗漏
4. 语言选择通过 localStorage 持久化，下次访问自动应用
5. 默认语言为英文，URL 无参数时显示英文

---

### US-16：表单字段和提示文字随语言切换

**描述：** 作为用户，我希望切换语言后，表单的标签、占位符、错误提示都使用对应语言，体验一致。

**验收标准：**
1. 投资意向表单（F2）和路条表单（F4）的所有标签、占位符、选项值、按钮文字均支持双语
2. 错误提示文字（如"此字段为必填"）随语言切换实时更新
3. 已填写的表单内容在语言切换后保留（不重置）
4. 邮件订阅（F8）的确认提示和订阅按钮文字均支持双语

---

## F8 — 市场洞察报告订阅

### US-17：订阅季度市场洞察报告

**描述：** 作为对新能源市场感兴趣的访客，我希望订阅 Inspira 的季度市场报告，获取行业洞察。

**验收标准：**
1. 页脚和 Insights 页面展示邮件订阅输入框和"订阅"按钮
2. 输入有效邮箱后点击订阅，显示"订阅成功，感谢关注"提示
3. 邮箱格式错误时显示红色错误提示
4. 订阅邮箱写入 Supabase subscribers 表，并同步至 SendGrid 邮件列表
5. 同一邮箱重复订阅时，显示"您已是订阅用户"提示，不重复写入

---

### US-18：收到订阅确认邮件

**描述：** 作为订阅者，我希望收到一封确认邮件，知道订阅成功并了解后续会收到什么内容。

**验收标准：**
1. 订阅后 5 分钟内，SendGrid 发送确认邮件至用户邮箱
2. 邮件包含：订阅确认、报告发布频率说明（季度）、Inspira 品牌 logo 和联系方式
3. 邮件底部含"取消订阅"链接，点击后从 SendGrid 列表移除
4. 邮件模板使用 Inspira 品牌配色，HTML 格式
