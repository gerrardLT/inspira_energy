## 详细模块说明

### 1. 问题（Problem）

**客户 A — 机构投资人面临的 3 个核心问题：**

| 问题 | 当前替代方案 | 不满意度 |
|------|------------|---------|
| 缺乏标准化新能源资产配置渠道 | 传统基础设施基金（透明度低）/ 直接收购电站（门槛高、$1亿+起） | 高 |
| 无法有效评估跨地域新能源项目风险 | 依赖第三方顾问（成本高、信息滞后） | 中 |
| 中小规模新能源资产难以触达 | 大型 PE 只做 $5000万+ 项目，中小项目缺乏机构级入口 | 高 |

**客户 B — 新能源开发商面临的 3 个核心问题：**

| 问题 | 当前替代方案 | 不满意度 |
|------|------------|---------|
| 项目融资渠道单一，依赖银行贷款 | 本地银行贷款（抵押要求高、周期长） | 高 |
| 路条持有成本高，变现渠道不透明 | 折价出售给单一买家 / 等待审批到期作废 | 高 |
| 难以接触国际资本，融资条款苛刻 | 本地 PE（条款苛刻、估值压低） | 中 |

---

### 2. 解决方案（Solution）

| 功能 | 解决的问题 | 为什么优于现有方案 |
|------|-----------|------------------|
| **四类标准化基金产品**（债权/股权/投建/COD） | LP 缺乏标准化入口 | 按风险偏好分层，投资人可自主选择进入节点，$500万起投 |
| **路条撮合机制** | 开发商路条变现不透明 | 建立行业路条信息枢纽，匹配买卖双方，降低信息不对称 |
| **全周期覆盖（投建→COD）** | 开发商融资渠道单一 | 符合标准共同投建，不符合帮助转让，开发商永远有退路 |

---

### 3. 独特价值主张（Unique Value Proposition）

**一句话 UVP：**

> **"Inspira 是唯一同时提供四类标准化新能源基金产品和路条流通撮合服务的跨境平台——让机构投资人零运营负担布局四大市场，让开发商永远有资金合作伙伴和路条退路。"**

**为什么选择 Inspira 而非替代方案：**

| 替代方案 | Inspira 的优势 |
|---------|---------------|
| Actis / Macquarie GIG | Inspira 专注 $500万-$5000万 中小项目，门槛低、响应快 |
| GCPC Group | Inspira 有标准化基金产品 + 路条撮合，GCPC 没有 |
| 传统银行贷款 | Inspira 无需抵押，通过基金结构分散风险 |
| 直接出售路条 | Inspira 撮合机制提高价格透明度，避免压价 |

---

### 4. 不公平优势（Unfair Advantage）

| 优势 | 竞品能否复制 | 说明 |
|------|------------|------|
| **路条撮合网络效应** | 难（需要双边网络） | 开发商越多 → 路条资源越丰富 → 吸引更多投资人 → 反过来吸引更多开发商 |
| **新加坡合规背书** | 中（可复制但成本高） | 新加坡注册地天然获得国际 LP 信任，中国大陆公司难以快速获得 |
| **跨境四地独家资源** | 难（关系驱动） | 同时在东南亚、澳洲、中国、欧洲有落地资源，单一地区竞争者难以覆盖 |
| **中小项目专精定位** | 中（大机构不屑做） | Actis/Macquarie 结构性不会下沉到 $500万 项目，形成战略真空 |

---

### 5. 客户细分（Customer Segments）

**客户 A — 机构投资人（优先级：高）**

| 属性 | 描述 |
|------|------|
| 画像 | 家族办公室、PE/VC、保险资金、主权基金 |
| 地区 | 新加坡、香港、澳大利亚、欧洲 |
| 投资规模 | $500万-$5000万 单笔，$1000万-$1亿 总配置 |
| 决策周期 | 3-6 个月 |
| 早期采纳者 | 新加坡/香港家族办公室（ESG 合规压力大，新能源配置需求迫切） |

**客户 B — 新能源开发商（优先级：高）**

| 属性 | 描述 |
|------|------|
| 画像 | 持有路条或在建项目的中小型开发商，员工 20-200 人 |
| 地区 | 中国、越南、印尼、泰国、菲律宾 |
| 项目规模 | 10MW-200MW，单项目价值 $500万-$8000万 |
| 痛点 | 融资难、路条变现慢、难以接触国际资本 |
| 早期采纳者 | 越南/印尼光伏开发商（市场热度高，路条供给充足） |

### 6. 渠道（Channels）

| 渠道 | 目标客户 | 优先级 | CAC 预估 |
|------|---------|--------|---------|
| **1v1 路演** | 机构投资人 | 最高 | 高（人力密集型） |
| **行业峰会**（Singapore Energy Week 等） | 机构投资人 + 开发商 | 高 | 中（品牌曝光+名片交换） |
| **官网 LP 意向漏斗** | 机构投资人 | 高 | 低（SEO + 内容驱动） |
| **LinkedIn 思想领导力** | 机构投资人 | 中 | 低（内容驱动） |
| **微信开发商社群** | 开发商 | 中 | 低（社群运营） |
| **行业协会渗透**（东南亚各国太阳能协会） | 开发商 | 中 | 中 |

---

### 7. 收入来源（Revenue Streams）

| 收入来源 | 模式 | 预估 |
|---------|------|------|
| **基金管理费** | AUM × 2%/年 | AUM $1亿 → 管理费 $200万/年 |
| **绩效分成** | 超额收益 × 20% | 基金 IRR 超 12% 部分提取 20% |
| **路条转让佣金** | 交易金额 × 3-5% | 单宗 $50万 路条 × 5% = $2.5万 |
| **项目咨询费** | 按项目计费 | 开发商咨询 $5-15万/项目 |

**单位经济模型：**
- 首年目标 AUM：$3000-8000万 → 管理费收入 $60-160万
- 第 3 年目标 AUM：$5亿 → 管理费收入 $1000万 + 绩效分成
- 路条撮合：第 1 年 5-10 宗，第 3 年 30-50 宗

### 8. 成本结构（Cost Structure）

| 成本项 | 第 1 年（万 USD） | 说明 |
|-------|----------------|------|
| 团队薪酬（6-8人） | 80-120 | 创始团队 + 开发 + 运营 |
| 法律/合规费用 | 20-30 | 新加坡 MAS 合规、基金文件 |
| 技术平台（官网+内部系统） | 10-15 | 开发+维护 |
| 差旅/市场拓展 | 15-20 | 东南亚/澳洲/欧洲路演 |
| 办公/行政 | 10-15 | 新加坡办公室 |
| **第 1 年总成本** | **$135-200万** | |

**盈亏平衡点：** AUM 达 $7000万-$1亿（约第 18-24 个月）

---

### 9. 关键指标（Key Metrics）

| 指标 | 定义 | 第 1 年目标 | 第 3 年目标 |
|------|------|-----------|-----------|
| **AUM** | 基金管理规模 | $3000-8000万 | $5亿 |
| **活跃项目数** | 正在投建/COD 的项目数 | 3-5 个 | 15-20 个 |
| **LP 转化率** | 官网意向提交 → 签约 | 5% | 15% |
| **路条撮合成交** | 撮合成功的路条交易数 | 5-10 宗 | 30-50 宗 |
| **基金 IRR** | 基金内部收益率 | 10-15% | 12-18% |

**北极星指标：** **AUM（基金管理规模）** — 最直接反映公司价值创造和收入规模

## 关键假设与验证优先级

| 假设 | 风险等级 | 验证方式 |
|------|---------|---------|
| LP 愿意通过新平台投入 $500万+ | 高 | 1v1 路演获取 Term Sheet |
| 开发商愿意在公开平台发布路条 | 中 | 先与 5 家开发商签订 MOU |
| 新加坡合规背书对欧澳 LP 有吸引力 | 中 | 参加 2 场欧洲/澳洲峰会收集反馈 |
| 路条撮合佣金模式可接受 | 中 | 与 3 家开发商讨论佣金比例 |
| 中小项目（$500万-5000万）有足够供给 | 低 | 行业数据验证 + 开发商访谈 |
